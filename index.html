about css avatars

  • This app was made just for fun as an exploration of CSS. Feel free to play around with and tweak the code. Check out the github repo here!
  • Responsiveness: For the most part, resizing the container div will resize your avatar, but there is an exception. Borders canâ€™t be set with % values, so certain features will look funky on resize. Any values not set in % are set in rems, so you can update those if you want to resize your avatar accurately.
  • There is some browser rendering funkiness that occurs only in mobile on transparent elements using rounded borders. You can learn more about the issue here. I am looking into alternatives, but in the meantime, I recommend using this app on desktop and/or screenshotting your creation.
  • I do plan on adding more appearance options in the future including: Facial expressions, more hairstyles, and accessories. If you have suggestions, let me know!
